<h2>介紹</h2>
<p>一個即時聊天室，需要輸入姓名及聊天內容，能看到對方是否在輸入中</p>
<h2>設計原理</h2>
<p>先設定socket,http連線，接著在server端儲存對話內容與用戶。在user端把對話的內容與用戶名稱印在畫面上</p>
<h2>使用方式</h2>
<p>打字聊天即可</p>
